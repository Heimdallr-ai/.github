### Microservices Architecture
```
Frontend (React)
    |
    v
AI-Backend (Proxy + ML)
    |
    +---> Auth Backend (Users, Auth, Subscriptions)
    |
    +---> Trading Backend (Trading Logic, Strategies)
```
